Guia de Estilo HTML

1. Nomenclatura de Classes

  • <div class="nome-da-classe"> - Use nomes descritivos que representem o propósito do elemento.
  • <div class="container"> - Utilize classes de layout genéricas como "container", "row" e "col" para criar uma estrutura flexível.
  • <div class="destaque"> - Reserve classes específicas para estilos de destaque.

2. Organização do Código

  • <header>, <nav>, <main>, <section>, <article>, <aside>, <footer> - Utilize elementos semânticos para estruturar o conteúdo.
  • <script> - Coloque scripts no final do corpo para garantir que a página seja renderizada primeiro.
  • <style> - Mantenha o estilo no cabeçalho do documento ou em arquivos CSS externos.

3. Comentários

  • <!-- Comentário --> - Utilize comentários para documentar partes importantes do código HTML.
  • <!-- Seção de cabeçalho -->
  • <!-- Rodapé da página -->
