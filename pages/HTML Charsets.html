Entendendo a tag HTML <meta charset>

Em HTML, a tag <meta charset> é usada para especificar a codificação de caracteres usada no documento.

A codificação de caracteres define como os caracteres em seu documento HTML são interpretados pelo navegador. O UTF-8 é uma codificação amplamente usada e suporta uma ampla gama de caracteres.

Incluir a tag <meta charset="UTF-8"> na seção <head> de sua página HTML garante que o navegador interprete corretamente os caracteres especiais e emojis.

Ao usar esses símbolos em texto HTML, é importante substituí-los por &lt; para < e &gt; para >. Isso é necessário para evitar que o navegador interprete erroneamente como parte do código HTML.

Aqui está um exemplo de como você pode incluir a tag <meta charset> em seu código HTML: 

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Minha Página</title>
        </head>
        <body>
            <h1>Olá, mundo!</h1>
            <p>Esta é a minha primeira página HTML.</p>
        </body>
        </html>
