<header>

O elemento <header> é usado para representar a introdução ou um grupo de conteúdo introdutório em uma página ou seção.

<nav>

O elemento <nav> é usado para representar uma seção de navegação em uma página. Ele geralmente contém links para outras páginas ou seções do site.

<section>

O elemento <section> é usado para representar uma seção autônoma de conteúdo em uma página. Ele agrupa conteúdo relacionado semanticamente.

<article>

O elemento <article> é usado para representar um conteúdo independente, como uma postagem de blog, um artigo de notícia ou um comentário.

<aside>

O elemento <aside> é usado para representar conteúdo relacionado ao conteúdo principal, como barras laterais, anúncios ou links relacionados.

<footer>

O elemento <footer> é usado para representar o rodapé de uma página ou de uma seção. Ele geralmente contém informações como informações de copyright, links para páginas relacionadas ou informações de contato.
