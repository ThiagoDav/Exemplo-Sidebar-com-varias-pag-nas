Tags HTML de Favicon com Exemplos

O favicon é o pequeno ícone exibido na barra de endereços do navegador, na guia da página e em outros lugares quando um site é visitado. Aqui está um exemplo de como adicionar um favicon a uma página HTML:

<link>

Para incluir um favicon em uma página HTML, você usa a tag <link> no cabeçalho do documento com o atributo rel definido como "icon" e o atributo href especificando o caminho para o arquivo de ícone. Por exemplo:

<link rel="icon" href="favicon.ico" type="image/x-icon">

Isso indica que o arquivo de ícone está localizado em "favicon.ico".

Recomendações para Favicon

  • É recomendável que o favicon tenha dimensões de 16x16 pixels ou 32x32 pixels.
  • O formato mais comum para o favicon é o formato ICO, embora outros formatos, como PNG, também sejam suportados pelos navegadores modernos.
  • O arquivo de favicon deve estar localizado na raiz do diretório do site para que os navegadores possam encontrá-lo facilmente.

Exemplo de Uso

Aqui está um exemplo prático de como incluir um favicon em uma página HTML:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Página</title>
    <link rel="icon" href="favicon.ico" type="image/x-icon">
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>
