Tag <table> em HTML

A tag <table> é usada em HTML para criar tabelas.

Exemplo de Uso:

Aqui está um exemplo simples de uma tabela HTML:

        <table>
            <tr>
                <th>Nome</th>
                <th>Idade</th>
            </tr>
            <tr>
                <td>João</td>
                <td>25</td>
            </tr>
            <tr>
                <td>Maria</td>
                <td>30</td>
            </tr>
        </table>

O código acima cria uma tabela com duas colunas: "Nome" e "Idade".

Resultado:

Nome Idade
João 25
Maria 30

Neste exemplo, cada linha da tabela é representada pela tag <tr>, e cada célula é representada pela tag <td> para dados comuns e <th> para cabeçalhos de coluna.
