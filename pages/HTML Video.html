Entendendo a Tag HTML <video>

A tag <video> é usada para incorporar conteúdo de vídeo em uma página HTML. Ela permite que você reproduza arquivos de vídeo diretamente no navegador.

Sintaxe Básica:

Aqui está um exemplo básico de como usar a tag <video> em HTML: 

        <video width="320" height="240" controls>
            <source src="video.mp4" type="video/mp4">
            Your browser does not support the video tag.
        </video>

Neste exemplo, estamos exibindo um vídeo com largura de 320 pixels e altura de 240 pixels. O atributo controls adiciona controles de reprodução padrão (play, pause, volume, etc.) ao vídeo. Se o navegador não suportar o vídeo, o texto "Your browser does not support the video tag." será exibido.

Atributos Importantes:

  • src: O atributo src especifica o URL do arquivo de vídeo.
  • width e height: Especificam a largura e altura do vídeo em pixels.
  • controls: Adiciona controles de reprodução ao vídeo, como play, pause e volume.
  • autoplay: Define se o vídeo deve começar a reproduzir automaticamente após o carregamento.
  • loop: Define se o vídeo deve ser reproduzido em um loop contínuo.
  • poster: Especifica uma imagem de visualização para ser exibida enquanto o vídeo está sendo baixado ou até que o usuário inicie a reprodução.

Além desses atributos, a tag <video> também suporta várias fontes de vídeo, permitindo que você forneça diferentes formatos de vídeo para garantir a compatibilidade com diferentes navegadores.
