Entendendo a Tag HTML <audio>

A tag <audio> é usada para incorporar conteúdo de áudio em uma página HTML. Ela permite que você reproduza arquivos de áudio diretamente no navegador.

Sintaxe Básica:

Aqui está um exemplo básico de como usar a tag <audio> em HTML: 

        <audio controls>
            <source src="audio.mp3" type="audio/mpeg">
            Your browser does not support the audio tag.
        </audio>

Neste exemplo, estamos exibindo um player de áudio com controles de reprodução padrão (play, pause, volume, etc.). Se o navegador não suportar o áudio, o texto "Your browser does not support the audio tag." será exibido.

Atributos Importantes:

  • src: O atributo src especifica o URL do arquivo de áudio.
  • controls: Adiciona controles de reprodução ao player de áudio, como play, pause e volume.
  • autoplay: Define se o áudio deve começar a reproduzir automaticamente após o carregamento.
  • loop: Define se o áudio deve ser reproduzido em um loop contínuo.

Além desses atributos, a tag <audio> também suporta várias fontes de áudio, permitindo que você forneça diferentes formatos de áudio para garantir a compatibilidade com diferentes navegadores.
