Entendendo as tags HTML < e >

Em HTML, os caracteres < e > são usados para representar os símbolos < e >, respectivamente.

O símbolo < é utilizado para iniciar uma tag HTML. Por exemplo, a tag <p> é usada para definir um parágrafo.

Da mesma forma, o símbolo > é utilizado para encerrar uma tag HTML. Por exemplo, a tag </p> é usada para fechar um parágrafo.

Ao usar esses símbolos em texto HTML, é importante substituí-los por &lt; para < e &gt; para >. Isso é necessário para evitar que o navegador interprete erroneamente como parte do código HTML.
