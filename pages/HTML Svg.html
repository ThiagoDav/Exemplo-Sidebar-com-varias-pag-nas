Entendendo a Tag HTML <svg>

A tag <svg> é usada para desenhar gráficos vetoriais escaláveis em HTML. SVG significa Scalable Vector Graphics (Gráficos Vetoriais Escaláveis) e é uma linguagem XML para descrever gráficos de forma bidimensional.

Ao usar a tag <svg>, você pode criar gráficos vetoriais, como formas geométricas, texto e caminhos, que podem ser escalados sem perda de qualidade.

Exemplo Básico:

Aqui está um exemplo básico de como usar a tag <svg> em HTML: 

        <svg width="100" height="100">
            <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
        </svg>

Este código cria um círculo vermelho em um espaço de desenho de 100x100 pixels.

Como Funciona:

Em SVG, você pode desenhar formas e aplicar estilos usando elementos e atributos XML. Por exemplo, o elemento <circle> é usado para desenhar um círculo, e os atributos como cx, cy e r especificam o centro e o raio do círculo.

Suporte do Navegador:

A tag <svg> é amplamente suportada pelos navegadores modernos, incluindo Chrome, Firefox, Safari e Edge. No entanto, é sempre uma boa prática verificar a compatibilidade com navegadores específicos ao implementar recursos de SVG em seu site.
