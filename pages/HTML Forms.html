Entendendo a tag HTML <form>

Em HTML, a tag <form> é usada para criar um formulário em uma página da web. Os formulários são usados para coletar informações do usuário, como nomes, endereços de e-mail, senhas, etc.

Um formulário geralmente contém vários elementos, como campos de entrada de texto, caixas de seleção, botões de opção e botões de envio.

Para enviar os dados do formulário para um servidor web para processamento adicional, você precisa especificar o atributo action na tag <form>. O atributo method especifica o método HTTP a ser usado para enviar os dados do formulário, como GET ou POST.

Ao usar esses símbolos em texto HTML, é importante substituí-los por &lt; para < e &gt; para >. Isso é necessário para evitar que o navegador interprete erroneamente como parte do código HTML.

Aqui está um exemplo de como você pode criar um formulário HTML básico: 

        <form action="/processar_dados" method="post">
            <label for="nome">Nome:</label>
            <input type="text" id="nome" name="nome"><br>

            <label for="email">Email:</label>
            <input type="email" id="email" name="email"><br>

            <input type="submit" value="Enviar">
        </form>

Este formulário tem campos para nome e e-mail, e um botão de envio. Quando o botão de envio é clicado, os dados do formulário são enviados para "/processar_dados" usando o método POST.
